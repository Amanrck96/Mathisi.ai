# Mathisi.ai - Better Version

Master the Art of Tech Intelligence with our industry-aligned programs.

## Features
- **Premium Design**: Modern Glassmorphism aesthetic with high-impact visuals.
- **8+ Specialized Programs**: Deep dives into AI, Data Science, Full Stack, and more.
- **Pay After Placement**: Focus on learning, not finance with our outcome-based model.
- **Expert-Led**: Content designed by industry veterans from Google, Amazon, and Meta.

## Tech Stack
- **React**: Modern component-based architecture.
- **Vite**: Ultra-fast development and build server.
- **Vanilla CSS**: Bespoke styling with dynamic variables and animations.
- **Framer Motion**: Smooth entrance and interaction animations.
- **Lucide React**: Clean, consistent iconography.

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the local development server.

## Deployment
Push to main to trigger automatic deployment (Vercel/Netlify recommended).
